# AGENTS.md

このリポジトリは「日ノ本異聞録：歴史の継承者」用のAndroidブラウザ向けPWAです。

## 方針
- API接続は使わない。
- OpenAI APIキーを保存しない。
- ゲーム状態はPWA側が管理する。
- ChatGPTは文章案だけを作る。
- 採用ボタンを押した本文だけ canonLog に保存する。
- AI返答をそのまま正史化しない。

## 実装方針
- 外部ライブラリなしのHTML/CSS/JavaScriptを優先する。
- Androidブラウザで読みやすいUIにする。
- 状態はJSONで管理する。
- セーブ/ロードを実装する。
- ローカル保存を使う。
- 後からFirefox Android拡張と接続しやすい関数構造にする。

## 禁止
- APIキーをコードに書かない。
- AIが世界状態を直接変更する設計にしない。
- 富士割を日常外出で常時携行扱いにしない。
- enemyState が NO-CONTACT の時に戦闘UIを出さない。
- 返信待ち状態なのに返信を自動発生させない。
- 旧境界へ現在地を巻き戻さない。

## テスト観点
- 初期画面が表示されること。
- state を保存・読み込みできること。
- AIプロンプトを生成できること。
- AI返答を貼り戻せること。
- 富士割・返信待ち・未接敵の検査警告が出ること。
